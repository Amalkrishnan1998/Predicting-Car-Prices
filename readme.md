# Predicting Car Prices

## Overview
This project predicts automobile prices using machine learning techniques.

## Dataset
IBM Automobile Dataset

## Tasks Performed
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Linear Regression
- Random Forest Regression
- Logistic Regression Classification
- Model Evaluation

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
